# Integrated Rental Pricing Tool for Property in Kuala Lumpur Malaysia #

This project focuses on the development of a data-driven rental pricing tool aimed at improving decision-making for both landlords and tenants in Kuala Lumpur, Malaysia. With rapid urbanization and rising property prices, tenants struggle to find affordable rental properties, while landlords face difficulty setting competitive rental prices. This tool leverages real-time market data and analytics to address these challenges, offering a streamlined solution for both parties.

Features:

Tenant Filters: Helps tenants find properties based on budget, features, and location using the 30% affordability rule.

Landlord Tools: Provides data-driven insights for setting competitive rental prices based on real-time market data.

Data Analytics: Descriptive, prescriptive, and diagnostic analytics to improve decision-making.

Property Insights: Personalized filtering, unit insights, and market condition comparisons.
